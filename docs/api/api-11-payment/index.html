    转账

    请求方式#

    POST请求，Content-Type为application/json

    Request header#

    字段长度备注
    X-token--用户登录后的token

    请求URL#

    /user-service/payment

    请求字段#

    字段类型长度示例可否为空说明
    receiverUsernamestring64---否接收用户名
    amountstring13---否转账数量(格式为5.00) 必须包含小数点和两位小数
    remarkstring------否备注
    typestring20---否目前固定为ONE_MONEY
    tradePasswordstring6---否资金密码

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg
    datadata----

    data返回内容#

    字段类型示例说明
    paymentNostring--转账单号

    请求示例：#

    {  "receiverUsername": "0123456789",  "amount": "100.00",  "remark": "用于转账",  "type": "ONE_MONEY",  "tradePassword":"090807"}

    返回示例：#

    {    "code": 200,    "data": {        "paymentNo": "1380005342397726720"    },    "msg": null}
