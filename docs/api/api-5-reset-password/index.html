    忘记/重置密码

    请求方式#

    POST请求，Content-Type为application/json

    1.发送验证码

    请求URL#

    /user-service/forget/password/send/code

    请求字段#

    字段类型长度示例可否为空说明
    usernamestring64"0123456789"否邮箱/或者手机号(手机号必须是0开头，且10位)

    2.重置密码，校验验证码

    请求URL#

    /user-service/forget/password/check/code

    请求字段#

    字段类型长度示例可否为空说明
    usernamestring64"0123456789"否邮箱/或者手机号(手机号必须是0开头，且10位)
    codestring6"4567"否验证码
    newPasswordstring12-20""否登录密码（必须同时包含英文字母、数字、特殊符号（!@#$%^&*））

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg

    验证码示例：#

    {    "username":"0123456789"
}

    返回示例：#

    {     "code": 200,    "msg": "ok"}

    2.重置密码，校验验证码示例：#

    {    "username":"0123456789",    "code":"0123456789",    "newPassword":"WE1qwqq01223!@#"
}

    返回示例：#

    {     "code": 200,    "msg": "ok"}
