    认证

    认证过程#

    1. 完成身份基本信息的填写
    2. 完成电子图片的上传
    3. 完成资金交易密码的设置

    资源文件#

    泰国区县市字典https://github.com/Cerberus/Thailand-Address

    请求方式#

    POST请求,Content-type为application/json

    Request header#

    字段长度备注
    X-token--用户登录后的token

    1.完成身份基本信息的填写#

    请求方式#

    GET请求

    请求URL#

    /user-service/complete/personal/info

    请求字段#

    字段类型长度示例可否为空说明
    nameString64---否姓名
    dateOfBirthString10"2020-10-10"否出生日期
    mobileString10---否0开头，手机号
    emailString-----否邮箱
    addressString-----否家庭住址
    provinceString-----否区
    districtString-----否县
    subDistrictString-----否市
    idTypeString-----否PASSPORT或ID_CARD
    idCardNumberString-------当idType为ID_CARD时。必须（必须是13为数字）
    idCardCodeString-------当idType为ID_CARD时。必须首字母必须是两位的大写字母，后面10位数字）如：AS0000000010
    passportNumberString-------当idType为PASSPORT时。必须护照号码）
    visaNumberString-------当idType为PASSPORT时。必须（签证号码）
    genderString固定内容Female可以固定内容, Female: 女，Male: 男, Non_disclosure: 不披露

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg

    返回示例：#

    {  "code":200,  "msg":"ok"}

    2.完成电子图片的上传#

    请求方式#

    POST请求,Content-type为application/json

    请求URL#

    /user-service/verify

    请求字段#

    字段类型长度示例可否为空说明
    idTypeString-----否认证类别:外国:PASSPORT;本地人:ID_CARD
    idCardFaceUrlString-------身份证正面照,类型为ID_CARD,必填
    idCardInHandUrlString-------手持身份证正面照,类型为ID_CARD,必填
    idCardBackUrlString-------身份证背面照，类型为ID_CARD,必填
    autographUrlString-----否电子签名
    passportHomeUrlString-------护照首页，类型为PASSPORT,必填
    passportVisaUrlString-------签证页，类型为PASSPORT,必填
    holdPassportUrlString-------手持护照，类型为PASSPORT,必填
    permitWorkUrlString-------工作证明，类型为PASSPORT,必填

    图片url的获取，系统提供了图片的上传功能，请看 图片上传 功能#

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg

    返回示例：#

    {"code":200,"msg":"ok"}

    3.完成资金交易密码的设置#

    请求方式#

    POST请求，Content-Type为application/json

    请求URL#

    /user-service/init/trade-password

    请求字段#

    字段类型长度示例可否为空说明
    passwordstring6"202106"否密码为纯数字

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg

    请求示例：#

    {    "password":"202106"}

    返回示例：#

    {     "code": 200,    "msg": "ok"}
