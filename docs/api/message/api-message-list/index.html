    消息列表查询

    请求方式#

    GET请求

    Request header#

    字段长度备注
    X-token--用户登录后的token

    请求URL#

    /user-service/message/list

    请求字段#

    字段类型长度示例可否为空说明
    cursorlong131628320978360可以为空消息开始时间戳，返回20条消息，为空时，返回截至当前时间到现在得20天数据

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg
    datadata----
    cursordata----

    data返回内容#

    字段类型示例说明
    titlestring--消息类型，前端须根据消息类型，显示不同的title以及国际化
    bodyobject--数据内容

    title 说明#

    字段说明
    PAY付款码成功时消息
    PAY_RECEIVE付款码付款成功时，收款方消息
    INCOME充值
    PAYMENT转账
    PAYMENT_RECEIVE转账收款方消息
    KYC_SUCCESS实名认证成功
    KYC_REJECT实名认证失败
    PAYMENT_TO_BANK转账到银行卡
    PAYMENT_TO_BANK_RETURN转账到银行卡被退回
    UTILITY生活缴费

    body说明#

    body中包含各种消息结构体：

    • 转账
    • 转账收款
    • 充值
    • 付款
    • 付款收款
    • KYC认证成功
    • KYC认证失败
    • 转账到银行卡
    • 转账到银行卡被退回
    • 生活缴费
    • 转账
    字段说明
    tradeNumber转账单号
    amount转账金额
    createTime发起时间
    finishedTime完成时间
    payer发起者
    payee接收者
    • 转账收款
    字段说明
    tradeNumber转账单号
    amount转账金额
    createTime发起时间
    finishedTime完成时间
    payer发起者
    payee接收者
    • 充值
    字段说明
    amount充值金额
    bankAccountName充值银行账户姓名
    bankAccountNumber充值银行账户账号
    createTime发起时间
    finishedTime完成时间
    type类型（BBL_QR:BBL通道,SCB_QR:SCB通道,OFF_LINE_CASH:线下柜台充值）
    tradeNumber单号
    • 付款方
    字段说明
    tradeNumber付款单号
    payer付款人账号
    payee收款人账号
    createTime发起时间
    finishedTime完成时间
    amount付款金额
    • 付款收款方
    字段说明
    tradeNumber收款单号
    payer付款人账号
    payee收款人账号
    createTime发起时间
    finishedTime完成时间
    amount收款金额
    • 实名认证成功
    字段说明
    name认证者姓名
    status认证状态
    msg认证状态说明
    email认证邮箱
    mobile认证手机号
    kycTime认证操作时间
    • 实名认证失败
    字段说明
    name认证者姓名
    status认证状态
    msg认证状态说明
    email认证邮箱
    mobile认证手机号
    kycTime认证操作时间
    • 转账到银行卡
    字段说明
    payerOneMoney账户
    tradeNumber转账单号
    amount数量
    bankName银行名称
    accountNumber银行卡号
    accountName银行卡持卡人姓名
    createTime操作时间
    finishedTime完成时间
    direction方向（IN:被退出转入，OUT:转出）
    • 转账到银行卡被退回
    字段说明
    payerOneMoney账户
    tradeNumber转账单号
    amount数量
    bankName银行名称
    accountNumber银行卡号
    accountName银行卡持卡人姓名
    createTime操作时间
    finishedTime完成时间
    direction方向（IN:被退出转入，OUT:转出）
    • 生活缴费
    字段说明
    tradeNumber交易单号
    createTime创建时间
    finishedTime完成时间
    type类型（UTILITY_NET：网费，UTILITY_WATER_ELECTRIC：水电费，UTILITY_MOBILE: 话费）
    totalAmount总缴费金额
    feeAmount手续费
    amount金额
    contentjson 字符串
    status状态（SUCCESS:完成，DOING:支付中，FAIL:失败，INIT:预下单）
