    TAG 29/30订单详情码

    TAG 29/30订单详情码#

    请求方式#

    GET请求

    /user-service/scan

    Request header#

    字段长度备注
    X-token--用户登录后的token

    扫码识别内容如下

    OM://tag2930/{xxxx订单号}

    请求服务获取码对应信息#

    字段类型示例说明
    codestring"OM://tag2930/1442338390392569856"扫码后获取到的qrCode,此码注意进行urlencode

    请求示例#

    /user-service/scan?code=OM%3A%2F%2Ftag2930%1442338390392569856

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg
    datadata----

    data返回内容#

    字段类型示例说明
    rsDtstring"1632465260332"交易时间
    rsUIDstring--银行单号
    qrPaymentNostring--系统单号
    feeAmountstring--手续费
    amountstring--金额
    statusstring--订单状态
    receiverProxyValuestring--接收者账号
    receiverMerchantNamestring--接收者姓名，该字段不是必传
    receiverBillerAccountNamestring--接收者账户名称
    ref1string--收款人单号
    ref2string--下单标示单号
    ref3string--下单标示单号
    sendNamestring--付款人姓名
    sendWalletIdstring--付款人账号
    errorMsgstring--msg
    paymentTypestring--交易类型
    payMethodstring--支付方式
