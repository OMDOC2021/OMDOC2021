    扫码状态说明 常见错误号#

    codemsg说明
    5300PAY_NUMBER_EXIST创建码时，系统生成单号重复，需重新请求
    5301PAY_STATUS_WRONG进行确认信息时，订单状态不符合时报错;例如扫码，订单状态必须为INIT，设置金额，订单状态必须为SCAN,付款订单状态必须为PAYING
    5302PAYER_DENY付款用户修改订单，越权。比如设置订单金额
    5303PAYEE_DENY收款用户修改订单，越权。比如确认支付
    5304PAY_EXPIRE订单超时，30分钟内操作有效
    5305CAN_NOT_SET_AMOUNT不能设置订单金额
    5306CAN_NOT_SCAN_YOURSELF不能自己扫自己
    5307PAY_NUMBER_NOT_EXIST查询时，或者修改订单状态时订单不存在
    5000INSUFFICIENT_BALANCE付款方余额不足以支付
