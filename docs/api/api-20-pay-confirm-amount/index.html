    设置收款金额

    请求方式#

    POST请求,Content-type为application/json

    Request header#

    字段长度备注
    X-token--用户登录后的token

    请求URL#

    /user-service/pay/confirm-amount

    请求字段#

    字段类型长度示例可否为空说明
    payNostring19"1399541764993449984"否订单号
    amountstring13"200.00"否200.00 比如包含2位小数

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg

    请求示例：#

    返回示例：#

    {    "code": 200,    "msg": "ok"}
