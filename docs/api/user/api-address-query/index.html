    用户地址查询

    请求方式#

    GET请求

    Request header#

    字段长度备注
    X-token--用户登录后的token

    请求URL#

    /user-service/user/address

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg
    dataobject--具体内容

    data说明

    字段类型说明
    address字符串详细住址
    province字符串区
    district字符串县
    subDistrict字符串市

    泰国身份证返回示例：#

    {  "code": 200,  "data": {    "address": "黑龙江xxx",    "province": "齐齐哈尔",    "district": "xx",    "subDistrict": "xxx"  },  "msg": "SUCCESS"}
