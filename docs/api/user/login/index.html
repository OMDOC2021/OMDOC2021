    登录/注册

    注：用户不存在，在进行校验验证码正确后会自动注册。 登录接口需要传用户设备Token，需要接入FCM

    1.用户密码登录

    请求URL#

    /login

    请求方式#

    POST请求，Content-Type为application/json

    header#

    字段长度备注
    deviceToken--app获取的deviceToken,可为空

    请求字段#

    字段类型长度示例可否为空说明
    usernamestring64"0123456789"否邮箱/或者手机号(手机号必须是0开头，且10位)
    passwordstring12-20---否登录密码

    2.验证码登陆

    2.1 请求获取验证码的URL#

    /user-service/sms/code

    请求方式#

    GET

    请求字段#

    字段类型长度示例可否为空说明
    usernamestring64"0123456789"否邮箱/或者手机号(手机号必须是0开头，且10位)

    2.2 用户名验证码登录#

    /user-service/sms/login

    请求方式#

    POST请求；Content-Type为x-www-form-urlencoded 或 POST请求，Content-Type为application/json

    header#

    字段长度备注
    deviceToken--app获取的deviceToken,可为空

    请求字段#

    字段类型长度示例可否为空说明
    usernamestring64"0123456789"否手机号(手机号必须是0开头，且10位)
    smsCodestring6---否验证码

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg
    datadata----

    data返回内容#

    字段类型示例说明
    tokenstring"39:0e0e9cf6-c9b8-4736-8834-8a831c6c1eb0"token
    needPasswordBooleanfalse是否设置登录密码
    verifystring"UNDO"用户状态
    expireTimeLong"1678734859487"时间戳
    idTypestring"1245648978477"用户标示"ID_CARD"（本地人）/"PASSPORT"（外国人）
    typestring"P"或"M"P: 个人， M: 商户

    用户名密码登录示例：#

    {    "username":"0123456789",    "password":"OMqwer10091!@#"
}

    返回示例：#

    {     "code": 200,    "msg": "Login Successfully",    "data": {      "token": "1:10f44a88-dd70-4701-8d66-9025a59f2c92",      "needPassword": false,      "verify": "PASS",      "expireTime": 1624689891387,      "idType": "ID_CARD",      "type": "P"     }}

    验证码请求示例：#

    {    "mobile":"0123456789"}

    返回示例：#

    {     "code": 200,     "msg": "SUCCESS"}

    手机验证码登录示例#

    {    "mobile":"0123456789",    "code":"717324"}

    返回示例：#

    {     "code": 200,    "msg": "Login Successfully",    "data": {       "token": "1:10f44a88-dd70-4701-8d66-9025a59f2c92",       "needPassword": false,       "verify": "PASS",       "expireTime": 1624689891387,       "idType": "ID_CARD",      "type": "M"    }}
