    1.充值方式

    请求方式#

    GET请求

    请求URL#

    /user-service/income/type

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"SUCCESS"code说明
    datajson--充值方式列表

    返回示例#

    {    "code": 200,    "data": [        {            "icon": "https://one-money-images.obs.ap-southeast-2.myhuaweicloud.com:443/c3229842e48141dca4c6c3bc182e2362.png",            "type": "SCB_QR",            "name": "Top up by Thai QR"        }    ],    "msg": "SUCCESS"}

    2.充值

    请求方式#

    POST请求，Content-Type为application/json

    Request header#

    字段长度备注
    X-token--用户登录后的token

    请求URL#

    /user-service/income/qr

    请求字段#

    字段类型长度示例可否为空说明
    amountstring13---否转账数量(格式为5.00) 必须包含小数点和两位小数

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"SUCCESS"code说明
    datajson--二维码内容

    data说明#

    字段类型示例
    qrImagestringbase64图片内容
    typestring"SCB_QR"
    incomeNumberstring"R1420238225359896576"

    {  "amount": "5.00"}
