    水费电费

    缴费充值流程#

    • 获取水电费充值信息
    • 预下单
    • 确认支付

    获取水电费充值信息#

    请求方式#

    GET请求

    /user-service/utility/water-electric/info

    返回内容#

    {  "msg": "操作成功",  "code": 200,  "data": {    "payments": [      "One Money"    ],    "notice": "ค่าธรรมเนียมในการเติมเงิน 20 บาท/รายการ"  }}

    预下单#

    请求方式#

    POST请求

    /user-service/utility/pre-order/water-electric

    请求字段#

    字段类型长度示例可否为空说明
    customerIdstring9---否客户ID
    meterNumberstring8---否Meter Number
    amountstring"0.00"---否缴费金额
    carrierstring"Metropolitan Electric"---否充值信息中的type的name字段

    请求示例#

    {  "customerId":"123456789",  "amount":"10.00",  "meterNumber":"12341234",  "carrier":"Metropolitan Electric"}

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg
    ------------
    depositAmountstring"10.00"缴费金额
    totalAmountstring"30.00"总金额
    orderNumberstring"1433649501930913792"单号
    meterNumberstring"12341234"Meter Number
    feestring"20.00"手续费
    customerIdstring"123456789"客户ID

    返回示例#

    {    "msg": "ok",    "code": 200,    "data": {        "depositAmount": "10.00",        "totalAmount": "30.00",        "orderNumber": "1433649501930913792",        "meterNumber": "12341234",        "fee": "20.00",        "customerId": "123456789"    }}
