    服务商接入 - 下单（app无需接入）

    请求方式#

    POST请求，Content-Type为application/json

    请求URL#

    /api/create-order

    请求字段#

    字段类型长度示例可否为空说明
    opTradeNumberString12-22位---否接入方单号
    merchantTradeNumberString64---是接入方商户订单号
    appidString64---否接入方appid
    merchantNumberString64---否接入商户号
    amountLong64---否下单金额,如下单1THB, 则需传100
    remarkString64---否备注
    timestampString64---否下单时间戳
    notifyUrlString64---否异步通知地址
    goodsString64---否商品名称
    signString64---否签名

    请求内容示例#

    {  "amount" : 2100,  "appid" : "nudh1n2eand12",  "goods" : "一个大苹果",  "merchantNumber" : "112233445566",  "merchantTradeNumber" : "MO-99889920019230124",  "notifyUrl" : "http://localhost:8889/",  "opTradeNumber" : "1122333445566778992",  "remark" : "test 二维码收款",  "sign" : "36e2c4dd5f92105a3b3bbf61761f271e6a426776b7e9d5a625cc64e7cfc5de55",  "timestamp" : 1632543513724}

    返回内容#

    字段类型示例说明
    codeString"200"200标示请求成功
    msgString"ok"msg
    datadata----
    ----------------
    merchantNameString----商户名称
    orderNumberString----OneMoney订单号
    statusString----订单状态(SUCCESS:成功, INIT:已下单, FAIL: 失败)
    merchantNumberString----商户单号(请求为传递，则为空)
    opNumberString----接入方单号
    goodsString----商户名称
    amountString----下单金额
    remarkString----用户备注
    expireTimeString----过期时间(15分钟内有效)
    createTimeString----创建时间
    updateTimeString----更新时间
    qrCodeString----二维码内容
    qrImageString----二维码图片流
    currencyString----币种名称(目前只有THB)
