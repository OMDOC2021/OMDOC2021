    OCR 功能

    请求方式#

    POST请求，Content-Type为application/json

    Request header#

    字段长度备注
    X-token--用户登录后的token

    请求URL#

    /user-service/ocr

    请求字段#

    字段类型长度示例可否为空说明
    typestring固定内容---否ID_CARD:泰国身份证 或PASSPORT: 护照
    urlstringhttps://support.huaweicloud.com/intl/zh-cn/api-ocr/zh-cn_image_0288050121.png---否首先需要将身份证或护照通过图片上传功能，获取服务器上图片URL

    返回内容#

    字段类型示例说明
    codestring"200"200标示请求成功
    msgstring"ok"msg

    身份证字段说明#

    字段类型说明
    sidestring标示正面还是反面，取值为front或back。
    id_numberstring身份证号。
    name_thstring泰文名字
    first_name_enstring英文名字
    last_name_enstring英文姓氏
    date_of_birth_thstring泰文出生日期
    date_of_birth_enstring英文出生日期
    religion_thstring宗教
    address_thstring地址
    date_of_issue_thstring泰文签发日期
    date_of_issue_enstring英文签发日期
    date_of_expiry_thstring泰文有效期
    date_of_expiry_enstring英文有效期
    serial_numberstring序列号
    card_numberstring身份证反面卡号
    laser_numberstring激光码
    confidencestring相关字段的置信度信息，取值范围0~1。置信度越大，表示本次识别的对应字段的可靠性越高，在统计意义上，置信度越大，准确率越高。 置信度由算法给出，不直接等价于对应字段的准确率
    portrait_imagestring头像的base64编码。当输入参数“return_portrait_image”为“true”时，才返回该参数。
    portrait_locationArray of objects头像在原图上的位置
    idcard_typestring身份证的类型。取值如下所示：+ normal：身份证原件 + copy是复印的身份证,当输入参数“return_idcard_type”为“true”时，才返回该参数

    护照字段说明#

    字段类型说明
    passport_typeString护照类型。P:普通因私护照 W:外交护照 G:公务护照
    country_codeString护照签发国的国家码
    passport_numberString护照号码
    nationalityString护照持有人国籍
    surnameString姓
    given_nameString名字
    sexString性别
    date_of_birthString出生日期
    date_of_expiryString护照有效期
    date_of_issueString护照签发日期
    place_of_birthString出生地
    place_of_issueString签发地
    issuing_authorityString签发机关
    confidenceString相关字段的置信度信息，取值范围0~1。置信度越大，表示本次识别的对应字段的可靠性越高，在统计意义上，置信度越大，准确率越高。 置信度由算法给出，不直接等价于对应字段的准确率。
    extra_infoString默认为空。对于中国护照，“extra_info”内会包含护照上由汉字描述的字段信息，如姓名、出生地等信息
